
## Getting Started

Clone the repository

```bash
git clone https://github.com/aabhiskarkc-max/serena-moon-fullstack.git
```

Go to project folder

```bash
cd serena-moon-fullstack
```

Go to Frontend 
```
cd serenamoon
```

Install dependencies

```bash
pnpm install
```

Start development server

```bash
pnpm dev
```

Build for production

```bash
pnpm build
```

Start production server

```bash
pnpm start
```

For Backend
```
cd serenamoon-api

```

Install dependencies
```
pnpm install
```
Stat the server 
```
pnpm start:dev
```


Get help from Nest CLI

```bash
nest -h
```

### Create Module

```bash
nest g module <module_name>
```

### Create Controller

```bash
nest g controller <controller_name>
```

### Create Service

```bash
nest g service <service_name>
```

### Create DTO

```bash
nest g d <dto_name>
```

---

## Requirements

```
node >= 18
pnpm
nestjs/cli
```

Install Nest CLI if not installed

```bash
npm install -g @nestjs/cli
```
